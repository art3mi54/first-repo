                                        Rock Paper Scissors

A simple command-line Rock, Paper, Scissors game written in Python. The player chooses a move and plays against a computer that selects randomly.

    Controls
Number	    Move
1	        Rock
2	        Paper
3	        Scissors

Requirements

Python 3.x

Running the Game
python rock_paper_scissors.py


Gameplay

- Enter a number (1–3) to select your move.

- The computer randomly selects its move.

- The program determines the winner.

- Choose Y to play again or N to exit.
