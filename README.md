# Rock Paper Scissors

A simple command-line Rock, Paper, Scissors game written in Python. The player chooses a move and plays against a computer that selects moves at random.

This small project is ideal for beginners learning Python, control flow, and basic CLI input/output.

-Simple command-line gameplay

-Random computer opponent

-Replay option (play again Y/N)

-Minimal dependencies (only Python 3)

- Python 3.x
- Standard library only (random, sys, etc.)

Running the Game

From the project directory, run:

```bash
python rock_paper_scissors.py
```

If your system uses python3 as the executable name, run:

```bash
python3 rock_paper_scissors.py
```

Gameplay

Controls table:

| Number | Move     |
|--------|----------|
| 1      | Rock     |
| 2      | Paper    |
| 3      | Scissors |

Example session (user input in bold):

```text
Welcome to Rock Paper Scissors!
Choose your move:
1) Rock
2) Paper
3) Scissors
> **1**
You chose Rock. Computer chose Scissors.
You win!
Play again? (Y/N) > **N**
Goodbye!
```

Notes:
- Enter 1–3 to select a move.
- The computer selects a random move.
- The program determines the winner and asks whether to play again.
